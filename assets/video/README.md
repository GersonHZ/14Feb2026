# Carpeta de Videos

Coloca aquí tus videos románticos para la página web.

## Archivos recomendados:
- `romantic-video.mp4` - Video principal (formato recomendado)
- `romantic-video.webm` - Formato alternativo para mejor compatibilidad

## Características del video:
- **Duración:** Entre 10-30 segundos (bucle infinito)
- **Resolución:** 720p o menor para carga rápida
- **Sin audio:** El video se reproduce silenciado
- **Formato:** MP4 (H.264) para mejor compatibilidad

## Consejos:
- Mantén el archivo de video pequeño (< 5MB) para carga rápida
- Usa videos en bucle que se vean bien repitiéndose
- Considera la privacidad al subir a GitHub (público)
